# Chapter 18: Viewing File Contents

- `cat file.txt` ā†’ Show entire file
- `less /var/log/syslog` ā†’ Scrollable view
- `more file.txt` ā†’ Pager
- `head app.log` ā†’ First 10 lines
- `head -20 app.log` ā†’ First 20 lines
- `tail app.log` ā†’ Last 10 lines
- `tail -f /var/log/nginx/access.log` ā†’ Real-time logs monitoring
