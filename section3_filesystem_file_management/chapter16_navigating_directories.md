# Chapter 16: Navigating Directories

## Absolute vs Relative Path
- Absolute → starts from `/`
 (e.g., `/home/shreeya/report.txt`)

- Relative → from current directory 
(e.g., `Documents/report.txt`)

## Commands
- `pwd` → Print current directory
- `cd` → Change directory
  - `cd` or `cd ~` → Home
  - `cd /etc` → Specific directory
  - `cd ..` → One level back
  - `cd ../..`→ Two level back 
  - `cd -` → Previous directory
