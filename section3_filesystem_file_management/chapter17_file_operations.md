# Chapter 17: File Operations

## Listing Files
- `ls` → List
- `ls -l` → Long format
- `ls -a` → Hidden files
- `ls -lh` → Human-readable sizes
- `ls -R` → Recursive listing

## Creating Directories
- `mkdir projects`
- `mkdir -p aws/ec2/nginx`

## Removing Directories
- `rmdir test` → Empty
- `rm -r test` → Non-empty
- `rm -rf test` → Force delete ⚠️

## Creating Files
- `touch file.txt`
- `touch app.log error.log`

## Copying Files
- `cp file.txt backup.txt`
- `cp -r project backup_project`

## Moving & Renaming
- `mv report.txt Documents/`
- `mv old.txt new.txt`

## Deleting Files
- `rm notes.txt`
- `rm file1 file2`
- `rm -r logs`
