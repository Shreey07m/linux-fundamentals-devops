# Chapter 3: History of Linux

## Before Linux
- 1970s → Unix dominates universities/enterprises.  
- Proprietary and expensive.  

## GNU Project (1983)
Richard Stallman launched GNU to create a free Unix-like OS. GNU provided tools but lacked a kernel.

## Linux Kernel (1991)
Linus Torvalds created the Linux kernel. Combined with GNU tools → GNU/Linux OS.

## Evolution
Linux rapidly expanded due to global contributions. Today it powers:
- Most web servers  
- Cloud platforms  
- Supercomputers  
- Android devices
