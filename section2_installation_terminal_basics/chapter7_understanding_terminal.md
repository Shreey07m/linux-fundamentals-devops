# Chapter 7: Understanding the Terminal

## What is a Terminal?
A program to interact with Linux using text commands.

Example:
- Windows: Click → Recycle Bin
- Linux: `rm file.txt`

## Why DevOps Engineers Prefer Terminal
Managing 500+ servers → automation via commands, not clicks.

## Terminal vs Console
- **Console** → Physical keyboard/monitor.
- **Terminal** → Software (GNOME Terminal, Windows Terminal).

## Opening Terminal
Ubuntu shortcut: `Ctrl + Alt + T`

## Prompt Breakdown
`shreeya@ubuntu:~$`
- `shreeya` → user
- `ubuntu` → hostname
- `~` → home directory
- `$` → regular user
- `#` → root user
