# Chapter 9: Bash

Bash = Bourne Again Shell. Default shell in most distros.

## Responsibilities
- Reads commands
- Executes programs
- Runs scripts
- Supports loops & variables
- Automation

## Examples
```bash
echo Hello
```
Output: ``Hello``

Check current shell: 
```bash 
echo $SHELL
```
Output: ``/bin/bash``

Available shells:
```bash
cat /etc/shells
```
