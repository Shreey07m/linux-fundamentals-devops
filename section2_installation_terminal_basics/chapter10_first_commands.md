# Chapter 10: First Linux Commands

- `pwd` → Print working directory  
- `ls` → List files  
- `ls -l` → Long listing  
- `ls -a` → Show hidden files  
- `clear` → Clear terminal (shortcut: Ctrl+L)  
- `whoami` → Current user  
- `hostname` → Computer name  
- `date` → Show date  
- `cal` → Calendar  
- `uname` → System info  
- `uname -a` → Kernel version, architecture
