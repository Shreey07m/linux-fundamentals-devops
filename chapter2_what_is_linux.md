# Chapter 2: What is Linux?

Linux is an open-source, Unix-like operating system built around the Linux kernel (created by Linus Torvalds in 1991).

## Why Linux is Popular
- Free and open source  
- Stable and reliable  
- Secure  
- Customizable  
- Ideal for automation and scripting  
- Preferred for cloud and DevOps  

## Linux vs Windows
| Feature | Linux | Windows |
|---------|-------|---------|
| Cost | Free | Paid license |
| Source Code | Open | Closed |
| Security | Strong | More malware |
| Customization | High | Moderate |
| Primary Use | Servers, Cloud | Desktop, Gaming |
